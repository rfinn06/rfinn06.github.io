Classses Vs. IDs

When to Use What

8/09/15

There are two different CSS selectors that can be used to save a developer time. These two selectors are "Class" and "ID". We will explore both of these in this article and give examples of when and how to use them.

Classes

A class can be used to grab a group of HTML tags and apply the same features to all of them in the group. What if we want to just apply changes to one set of h1 tags in our HTML file and not the others? To do this you can use IDs.
