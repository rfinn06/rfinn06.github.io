Enumerable Methods

Specifcally Map

8/23/15

Enumerable is a module inside Ruby that houses built in methods that can be used by many unrelated classes. The mothods housed in the Enumerable module can only be used by classes that have an each method define. Arrays and Hashes have an each method defined, therefor they can use Enumerable methods. There are many Enumerable methods available for Rubysists to use. In this article I will explore the map method.

The map method will work through a collection (Array or Hash) and perform a block of code on each item to produce a new array. Let's say you have an array that stores ages in a database. This database has not been updated in two years so we need to update it. We can use the map methods to add two years to each age listed in the array.

my_ages = [2, 34, 67, 9, 17]

my_ages.map { |num| num + 2 }
=> [4, 36, 69, 11, 19]

As you can see, the map method is powerful tool that can be used to make many different transformations of arrays and hashes. The map method is just one example of the Enumberable methods that are available. You can explore other Enumerable methods in the Ruby Docs
