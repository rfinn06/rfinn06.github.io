Building a Ruby Class

8/30/15

Ruby

Classes in Ruby are well suited to modeling the real world. In a class you can create new objects that have specified chracteristics and methods. In this post I will build a class that models pizza. Yes, pizza.

class Pizza

  def initialize
    @size = size
    @toppings = toppings
    @cheese = cheese
  end

  def size
    @size = ["Small", "Meduim", "Large"]
  end

  def toppings
    @toppings = ["Mushrooms", "Onions", "Green Peppers", "Olives", "Pepperoni", "Sausage"]
  end

  def cheese
    @cheese = ["Mozzarella", "American", "Cheddar"]
  end

  def suggested
    suggested_size = @size.sample
    suggested_toppings = @toppings.sample(3)
    suggested_cheese = @cheese.sample
    puts "Your suggested pizza is a #{suggested_size} with #{suggested_toppings} and #{suggested_cheese} cheese!"
  end

end

pizza = Pizza.new

pizza.suggested

So we created the class and initialized it with certain attributes. In addition to defining what the attributes of our pizza class are, we also created a method to make use of these attributes. I enjoy eating pizza, but sometimes have a hard time deciding what to get on it. To solve this we create a method that will create a suggestion for the type of pizza to get. The method "suggested" will call upon the attributes of the pizza class and build us a suggested pizza to order. My life just got a little bit easier.
